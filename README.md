# Instruciones:

1. **Paso 1** Clonar el repo e instalar los dependencias necesarios:
  ```
  pip -r requirements.txt
  pip install django==1.10.1
  ```

2. **Paso 2** Ejecutar:
  ```
  python manage.py runserver
  ```
